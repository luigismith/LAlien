## Come aggiungerli

1. Salva le PNG dal dispositivo in questa cartella con i nomi sopra
2. Dalla root del repo:
   ```bash
   git add docs/screenshots/*.png
   git commit -m "docs: add gameplay screenshots"
   git push
   ```

Gli screenshot appariranno automaticamente nel README principale.
